# Sistema de Bombagem – C#

Este projeto é um exercício académico desenvolvido em C# que
simula um sistema simples de bombagem, envolvendo bombas e válvulas.

## Descrição
O sistema é composto por uma classe base `Equipamento` e pelas
classes derivadas `Bomba` e `Valvula`.

Cada equipamento possui estados que podem ser controlados através
dos métodos:
- `ligar()` / `desligar()`
- `abrir()` / `fechar()`

## Funcionalidades
- Ligação e desligamento de bombas
- Abertura e fecho de válvulas
- Apresentação do estado dos equipamentos
- Simulação de arranque e paragem do sistema

## Conceitos aplicados
- Programação Orientada a Objetos
- Herança
- Polimorfismo
- Métodos virtuais e sobrescrita (`virtual` / `override`)
- Arrays de objetos

## Tecnologias
- Linguagem: C#
- Tipo: Aplicação Console (.NET)
